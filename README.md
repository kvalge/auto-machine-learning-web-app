# Auto Machine Learning Web App

### Functionalities
- Upload data  
- Get summary report of uploaded dataset  
- Test and compare classification, regression or clustering models   
- Download best model  
- [The best model info](model_info.txt)  

### Libraries  
Streamlit  
Pandas  
YData Profiling  
Streamlit Pandas Profiling  
Pycaret
